## Why Polestar OSS?

Our tools are built by the community, for the community. Every application prioritizes your privacy and data ownership.

### Privacy-First
- 100% client-side processing
- No backend servers
- No data collection
- No tracking
- All processing happens locally in your browser

### Open Source
- MIT licensed
- Community-driven
- Transparent development
- Contributions welcome
